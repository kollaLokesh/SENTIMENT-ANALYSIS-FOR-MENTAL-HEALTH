# Sentiment Analysis for Mental Health in Telugu

This project explores mental health sentiment detection in Telugu-language text using NLP and machine learning techniques. The goal is to identify emotional patterns and potential indicators of distress in user-generated comments and related text data.

## Project Overview

The repository contains:

- a Jupyter notebook with the main analysis and model workflow
- a dataset used for training and experimentation
- a project report in PDF format
- an archive of the project materials

## Repository Contents

- [project.ipynb](project.ipynb) - main notebook for preprocessing, analysis, and model building
- [Final_dataset.xlsx](Final_dataset.xlsx) - dataset used in the project
- [Sentiment_Analysis_Telugu.pdf](Sentiment_Analysis_Telugu.pdf) - project report
- [sentiment analysis.zip](sentiment%20analysis.zip) - zipped project files

## How to Run

1. Open [project.ipynb](project.ipynb) in Jupyter Notebook or VS Code.
2. Install the required Python libraries if they are not already available in your environment.
3. Run the notebook cells from top to bottom.
4. Review the outputs for preprocessing steps, model results, and analysis.

## Notes

- The notebook filename in this workspace is [project.ipynb](project.ipynb). If you expected a file named `CODE.ipynb`, rename the notebook or update any references accordingly.
- The presentation and HTML report mentioned in the template are not present in this folder.
